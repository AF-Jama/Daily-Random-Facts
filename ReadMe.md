# Daily Random Facts 

A short Twitter bot that tweets/update status every 5 minutes
Short script uses two API's

- Twitter Api (Using Tweepy wrapper)
- Useless Facts API 

# Technologies/Libaries Used

- Python
- Requests
- Tweepy
- Json
- datetime
